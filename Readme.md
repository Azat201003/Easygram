# Easygram

It is TUI telegram client created for programmers. I want to add here much feutures.

## Prerequirements

I using free libraries in my code. You need to install next ones:

 - [TDLib](https://github.com/tdlib/td)
 - [logger](https://github.com/Azat201003/logger)

## Building

To build just write

``` bash
make bld
```

in root path.

## Running

To run builded just write

``` bash
make run
```
